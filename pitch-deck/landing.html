FLAGSHIP_PRODUCT::CLAWREFORM

clawREFORM
Deep Dive

The open-source Agent Operating System that can evolve itself. Three key innovations define the platform.

01

The Organ System

Agent context as discrete, human-editable, git-trackable files — not a prompt blob. Edit SOUL.md in any editor → personality changes instantly. Version in git → full audit trail.

02

The Memory Ladder

Four-tier memory with biological decay. Core, Overview, Project, and Working Detail. Important things persist — the rest fades naturally. Cross-channel canonical sessions.

03

Self-Modification Engine

ClawReform can rewrite its own Rust source code with a full safety pipeline: Analyze → Backup → Modify → Validate → Rollback → Approve. Human-in-the-loop optional.
