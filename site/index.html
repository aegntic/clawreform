ClawReform

The self-evolving Agent OS built in Rust

  • Self-modification kernel with safety sandboxing
  • 60+ bundled skills for common automation tasks
  • 23+ MCP server integrations (PostgreSQL, Notion, GitHub...)
  • Tailscale mesh networking for distributed agents
  • Multi-agent A2A protocol for agent-to-agent communication
  • Enterprise security with full audit logging

Terminal
# Clone and build
$ git clone https://github.com/aegntic/clawreform
$ cd clawreform && cargo build --release

# Start the agent daemon
$ ./target/release/clawreform start

# Create your first agent
$ curl -X POST http://127.0.0.1:4332/api/agents \
  -H "Content-Type: application/json" \
  -d '{"name": "DevOps Agent"}'

# Send a task
$ curl -X POST http://127.0.0.1:4332/api/agents/1/message \
  -d '{"message": "Deploy staging"}'
